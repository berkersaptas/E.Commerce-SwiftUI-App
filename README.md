# FluxStore E-Commerce App

SwiftUI project

# Features

 - Splash - Welcome Page
 - Intro - Onboarding Page
 - Create User Page
 - Login User Page
 - Forget Password Page
 - User Email Validate Page
 - Verification Code Page
 - Create New Password Page
